# Ember Setup

Installer and setup wizard for [Ember-2](https://github.com/niansahc/ember-2) — a local, private personal intelligence system.

## Development

```bash
npm install
npm start          # demo mode
npm run start:real # real commands
```

## Build

```bash
npm run build:win
npm run build:mac
npm run build:linux
```
